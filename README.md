# kajson

## Introduction

Powerful universal JSON encoder/decoder for Python objects with support for pydantic v2.

This code is used by [Pipelex](https://github.com/Pipelex/pipelex).

## Installation

### Using pip

```bash
pip install kajson
```

### Using poetry

```bash
poetry add kajson
```
